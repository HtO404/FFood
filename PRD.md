# FFood — 家庭食材管理 PRD

> iOS16 风格 · Vue 3 + Vite · 本地优先 · 2026-07-03

---

## 一、产品概述

FFood 是一款 iOS16 风格的**家庭食材管理** Web 应用（可封装为小程序）。
核心场景：用户购入食材后扫码/手动录入，系统按保质期自动提醒，避免浪费。

## 二、需求分级

| 编号 | 功能 | 验收标准 | 状态 |

## 三、技术架构

```
视图层 (View)             逻辑层 (Logic)           数据层 (Data)
┌─────────────────┐    ┌──────────────────┐    ┌──────────────┐
│ App.vue         │───▶│ foodStore.js     │───▶│ localStorage │
│ ├─ 导航栏       │    │ ├─ addFood()     │    │ "ffood_data" │
│ ├─ 搜索栏       │    │ ├─ updateFood()  │    └──────────────┘
│ ├─ 分类Chip     │    │ ├─ removeFood()  │
│ ├─ 食材卡片列表 │    │ ├─ load()        │
│ ├─ 添加/编辑弹窗│    │ └─ save()        │
│ └─ 删除确认弹窗 │    └──────────────────┘
└─────────────────┘
```

| 层 | 技术 |
|----|------|
| 框架 | Vue 3 (Composition API) |
| 构建 | Vite 6 |
| 样式 | iOS16 Design Tokens + 毛玻璃效果 |
| 状态管理 | 单例 FoodStore + reactive |
| 持久化 | localStorage |
