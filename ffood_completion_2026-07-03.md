## 技术栈

- **框架:** Vue 3 Composition API (单文件组件)
- **构建:** Vite 6，build 886ms
- **样式:** iOS 16 Design Tokens（毛玻璃/系统色/圆角/阴影/字体级联）
- **状态:** 单例 FoodStore + reactive + localStorage 持久化
- **架构参考:** CSDN 微信小程序双线程模式 → Vue 版映射
